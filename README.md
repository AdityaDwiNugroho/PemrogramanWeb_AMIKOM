## Daftar Praktikum

Kumpulan tugas dibagi ke dalam beberapa modul pertemuan yang masing-masing merangkum standar praktik web modern:

- **[Modul 1: Pengenalan HTML](https://pemrograman-web-amikom.vercel.app/pertemuan-1)**
  Berisi pemahaman dasar tag HTML (Typography, Lists, Images) serta tabel kompleks untuk manipulasi struktur baris dan kolom yang menggunakan `colspan` dan `rowspan`.

- **[Modul 2: Semantic Tag](https://pemrograman-web-amikom.vercel.app/pertemuan-2)**
  Berisi penerapan Document Structure yang mendeskripsikan arti kepada browser maupun developer menggunakan tag Semantic HTML5 (`<header>`, `<nav>`, `<article>`, `<section>`, `<footer>`). Modul ini juga menunjukkan penyematan video melalui YouTube `<iframe>`, struktur gambar dinamis dengan `<figure>`, serta elemen interaktif lewat `<details>` tag.

## Highlights & Fitur Terkini

- Aestetik **Modern & Bersih**: Menggunakan palet yang direkomendasikan, ikon SVG (Lucide Icons), *soft shadow drops*, elemen yang dikelompokkan dengan border radius, dan *hover effect* yang dinamis.
- **Routing Clean URLs**: Implementasi melalui *Vercel build engine* (`vercel.json`) guna memberikan tautan *path URL* bersih bebas dari `.html` sehingga lebih profesional dan *User-friendly*.
- **Fetch Otomatis**: *Beranda utama (root)* dibuat otomatis untuk melakukan *fetch* struktur folder `github repository` ini secara asinkron (API Github), menjadikan pengelolaan tautan-tautan per modul menjadi praktis.
